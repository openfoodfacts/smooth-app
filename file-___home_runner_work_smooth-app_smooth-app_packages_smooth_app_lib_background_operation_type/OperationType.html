OperationType

OperationType enum Null safety

Type of a transient operation.

We need a type to discriminate between operations (cf. getNewKey). When we are in a collection of operations, the keys will helper us know

  • which type (cf. matches)
  • which order (cf. getSequentialId, sort)
  • possibly, which barcode (not useful yet)
